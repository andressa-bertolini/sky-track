## 🛠️ Installation

1. Clone the repository:

```bash
git clone https://github.com/andressa-bertolini/sky-track
cd sky-track
```

2. Install dependencies:
```
npm install
```

3. Start the development server:
```
npm run dev
```

The app will run at http://localhost:3000
